Ninja Ranger Game Introduction

Ninja Ranger uses a realistic pixel style, bringing you back to the NES, 8-bit, 16-bit pixel game era; the game is also using the classic 2D platform action gameplay, this is an retro action game with a refreshing impact. It tells the story of a legendary shadow ninja--Ray, the shadow ninja punishes evil and save the world. In the game you will play a ninja superhero and fight against evil enemies. It is released on apple app store and google play store.
